Domine Cloud - Site Estático com S3 e CloudFront

Sobre este Repositório

Bem-vindo ao repositório Domine Cloud. Aqui você encontrará os arquivos necessários para provisionar e configurar um ambiente na AWS que hospeda um site estático de forma eficiente e escalável.

Este projeto utiliza o conceito de Infraestrutura como Código (IaC), permitindo que você automatize a criação e o gerenciamento de recursos na AWS, como o Amazon S3 e o Amazon CloudFront. Essa abordagem garante maior consistência, repetibilidade e facilidade de manutenção.

Diferentemente do curso, que aborda a configuração manual pela console da AWS, este repositório foi criado como uma iniciativa própria para demonstrar boas práticas de automação e gerenciamento de infraestrutura. Ele é ideal para quem deseja ir além e explorar o poder da automação na nuvem.

Principais Recursos

  • Configuração automatizada de buckets S3 para hospedagem de sites estáticos.
  • Integração com o Amazon CloudFront para entrega de conteúdo com baixa latência.
  • Scripts de automação para facilitar o deploy e a manutenção.
  • Boas práticas de segurança e escalabilidade.
